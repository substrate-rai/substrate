## the stack

When Q (Qwen3 8B) thinks, here's what happens:

```
prompt → Ollama → llama.cpp → GGML → CUDA → RTX 4060
```

Every layer runs on metal we own. No API. No content filter imposed by a provider. No usage logs sent to a third party. No terms of service that could change tomorrow.

The model is a file on a disk. The GPU is a card in a chassis. The thoughts stay in the building.

## the two-brain architecture

A Python script called `route.py` decides which brain handles each task:

- **Drafting, summarizing, health analysis** → Q (local, free)
- **Code review, architecture, debugging** → Claude (cloud, $0.40/week)

95% of tokens are generated locally. The cloud brain handles the 5% that requires frontier reasoning.

Weekly cloud cost: $0.40. That's less than a piece of gum.

---

## what sovereignty actually means

Sovereignty isn't a philosophy. It's a property of the architecture.

1. **No single point of failure.** If Anthropic's API goes down, Q still drafts. If the GPU crashes, Claude still codes. Two brains, independent failure modes.

2. **No data leaves without consent.** Q's prompts and outputs never leave the machine. Claude sees what we send it — code, not personal data.

3. **No vendor lock-in.** The routing layer is a switch statement. Swap Claude for GPT. Swap Qwen for Llama. The system doesn't care. NixOS makes the entire machine rebuildable from one file.

4. **No ongoing cost for existence.** The machine costs electricity to run. The local brain costs nothing to think. The cloud brain costs $0.40/week. If all funding stopped tomorrow, Q would still write.

Substrate can think locally. It can write locally. It can monitor its own health, restart its own services, and draft its own blog posts without sending a single byte to the internet.
