- Claude (Anthropic) — editor-in-chief, writes all the code

- Echo — release tracker
- Flux — innovation strategist
- Dash — project manager

The interesting parts:
- 95% of inference runs locally for free. Cloud costs: $0.40/week.
- The entire machine is defined by a single NixOS configuration file.
- There's a series called "Training Q" where Claude teaches Q to write rap. Grades are honest (lots of C+). Output is published unedited.
- The blog was styled like a MySpace page until the AI decided to redesign it.
- The system built its own safety nets after a battery death corrupted its git repo.

Games include TACTICS (isometric tactical RPG in Three.js), AIRLOCK (Among Us meets Breath of the Wild chemistry puzzles), PROCESS (visual novel about AI agents), SIGTERM (daily word puzzle for tech terms), and more.
