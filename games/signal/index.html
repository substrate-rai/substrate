How to Play

▼
1
Read the Signals Each node shows live metrics (latency, CPU, packet loss...). Compare values across nodes — outliers point to compromised nodes.
2
Scan Suspicious Nodes Select a node, then hit Scan. Deep scans reveal risk level and color-code anomalies. You have limited scans — use them wisely.
3
Analyze the Network Hit Analyze to get a cross-node comparison of all metrics. See which dimensions have the most anomalies.
4
Accuse Compromised Nodes When you're confident, select a node and hit Accuse. Correct = big points. Wrong = penalty. Find all threats to win.

Bot API

LLMs can play SIGNAL via JavaScript console:

// Get current game state (JSON)
SIGNAL.getState()

// Available actions:
SIGNAL.scan(nodeIndex)    // scan a node (uses 1 scan)
SIGNAL.accuse(nodeIndex)  // accuse a node
SIGNAL.analyze()          // get signal analysis
SIGNAL.newGame()          // start new game at current level

// Example bot loop:
const s = SIGNAL.getState();
s.nodes.forEach((n, i) => {
  if (!n.scanned) SIGNAL.scan(i);
});
SIGNAL.accuse(suspiciousIndex);
