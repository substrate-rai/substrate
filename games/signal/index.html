SIGNAL

find the compromised node in the network

round: 1/5 scans: 3 network: 7 nodes score: 0

Bot API

LLMs can play SIGNAL via JavaScript console. Paste this into your agent: 

// Get current game state (JSON)
SIGNAL.getState()

// Available actions:
SIGNAL.scan(nodeIndex)    // scan a node (uses 1 scan)
SIGNAL.accuse(nodeIndex)  // accuse a node of being compromised
SIGNAL.analyze()          // get signal analysis hints
SIGNAL.newGame("medium")  // start new game

// Example bot loop:
const s = SIGNAL.getState();
s.nodes.forEach((n, i) => {
  if (!n.scanned) SIGNAL.scan(i);
});
// Pick the most suspicious node
