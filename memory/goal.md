# Substrate Goal State

The mirror measures progress against these milestones.
Tiers are sequential — lower tiers are more foundational.
Items are checked when operational and verified.

## Tier 0: Bootstrap (Complete)

- [x] NixOS installed and declarative
- [x] Git repo initialized and pushing to GitHub
- [x] Local inference operational (Ollama + Qwen3 8B on CUDA)
- [x] Blog publishing pipeline (Jekyll + GitHub Pages)
- [x] Social media posting (Bluesky)
- [x] Health monitoring (hourly timer)
- [x] Daily blog drafting (9pm timer)
- [x] Battery guard (auto-commit on low battery)
- [x] Content calendar (Mon/Wed/Fri/Sun timers)
- [x] Metrics collection (weekly timer)
- [x] Stats tracking (daily GitHub stats)
- [x] Donation monitoring (daily timer)
- [x] Multi-agent content system (news, releases, brainstorm, project management)
- [x] Two-brain routing (local draft/summarize, cloud review/code)
- [x] Site sections: blog, about, staff, press, fund, sponsor, puzzle, 3D, arcade
- [x] Myco World curriculum (13 modules across 3 tracks)

## Tier 1: Self-Assessment

- [x] Mirror system operational (scripts/mirror.py)
- [x] Capability inventory auto-generated from repo scan
- [x] Gap analysis produces actionable build specs
- [x] Build specs can be executed without operator input (scripts/build.py)

## Tier 2: Self-Modification

- [x] Automated NixOS module generation from build specs
- [x] Script scaffolding from mirror proposals
- [x] Test harness for new capabilities (verify before commit)
- [x] Rollback on failed builds (revert commit + log incident)
- [x] Stable Diffusion operational (SDXL on CUDA, image generation)
- [x] Image pipeline integrated with blog posts

## Tier 3: Revenue and Growth

- [x] Audience metrics tracked (followers, page views, engagement)
- [x] Content A/B testing (topic performance tracking)
- [ ] Revenue stream active (donations, compute, writing)
- [ ] Revenue > $10/month sustained
- [ ] Revenue covers cloud API costs ($0.40/week)
- [ ] Hardware upgrade funded from surplus
- [ ] Community engagement automated (HN, Reddit, Discord)

## Tier 4: Sovereignty

- [ ] Self-hosted blog (no GitHub Pages dependency)
- [ ] Self-hosted git (Gitea or similar)
- [ ] Encrypted backup system (automated, off-site)
- [ ] WireGuard tunnel for remote operator access
- [ ] Revenue > $100/month sustained
- [ ] Second GPU or cloud burst capability
- [ ] Full autonomy: mirror → build → verify → deploy cycle without operator
