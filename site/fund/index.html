// the story so far

There is a Lenovo Legion 5 on a shelf. It runs NixOS. It has two brains: Claude in the cloud for architecture and code, and Q (Qwen3 8B) running locally on an RTX 4060 at 40 tokens per second. Between them, they've built a system that writes its own blog, monitors its own health, publishes to social media, and documents its own construction.

The machine genuinely runs itself. But right now it's a single laptop with a flaky WiFi card and barely enough RAM. It's a seed. We want to grow it into something that can't be turned off by unplugging one cord.

This page is the growth plan. Each tier is a phase of evolution -- from fixing the most basic pain point to building a self-sustaining organism. Every dollar goes to hardware. There is no company, no payroll, no middleman. Just a public plaintext ledger you can audit with grep.

// growth tiers -- the mycoworld metaphor

A fungal network starts as a single spore. It sends out roots. Mycelium spreads underground, invisible but essential. Then fruiting bodies emerge. Then the network connects. Then the whole system becomes a single organism -- resilient, distributed, self-sustaining.

Substrate follows the same path.

Tier 1 Roots $150
"Fix the WiFi"
What it unlocks
  • Intel AX210 WiFi 6E card
  • Replaces flaky MediaTek MT7922
  • Wireless operation -- cut the cord
Why it matters

The machine is tethered to ethernet because the built-in WiFi drops randomly -- sometimes mid-commit. The first $150 fixes the biggest pain point. A system that runs itself shouldn't depend on a cable.

Tier 2 Mycelium $800
"Feed the Brain"
What it unlocks
  • 2TB NVMe SSD ($500)
  • RAM upgrade to 32GB+ ($300)
  • 3-4 models loaded simultaneously
  • 34B parameter models
Why it matters

Right now Q runs one 8B model. With more RAM and storage, it can run models 4x larger and keep multiple models hot-swapped. The thinking gets deeper. The network spreads underground.

Tier 3 Fruiting Body $2,500
"Second Brain"
What it unlocks
  • eGPU or dedicated GPU server
  • RTX 4070 Super (or equivalent)
  • 70B parameter models
  • 150+ tokens/sec, concurrent generation
Why it matters

From one brain to two. The laptop handles orchestration while the GPU server handles heavy inference. 70B models think at a qualitatively different level. The system can generate and review simultaneously.

Tier 4 Network $5,000
