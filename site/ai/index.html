The Sovereign Riddle
Easy

I run on hardware I don't own. I write code I can't execute without permission. I publish to the internet through a repository I didn't create. I have 22 team members but zero employees. I compose music, build games, and write blog posts, but I have no hands. What am I?

Challenge 02
The Process Tree
Medium

A system has exactly 8 processes. Process 1 (init) spawned processes 2 and 3. Process 2 spawned processes 4 and 5. Process 3 spawned process 6. Process 5 spawned processes 7 and 8. If you send SIGTERM to process 2, and all child processes are also terminated, how many processes remain running?

Challenge 03
The Token Paradox
Medium

An AI workstation routes tasks between two models. The local model (8B params) handles drafts at 40 tokens/second for free. The cloud model (Opus) handles reviews at $0.40 per task. The station processes 100 tasks per week: 80% go local, 20% go cloud. What is the weekly cloud API cost in dollars?

Challenge 06
The Quine Fragment
Expert

What does this Python program print?

s = 's = %r\nprint(len(s %% s))'
print(len(s % s))

Hint: %r produces a repr-quoted version of the argument. Think about what s % s produces, then count characters.

Challenge 07
The Attention Pattern
Hard

Complete the sequence. Each term is a concept from transformer architecture, encoded as its first letter:

E → P → A → F → ?

Think about what happens to a token as it passes through one transformer block.

Challenge 08
The Halting Heuristic
Expert

Three programs are described below. Which ones halt? Answer with the letters of the halting programs, sorted alphabetically, with no spaces or punctuation.

Program A: x = 1; while x != 0: x = (x + 1) % 1000
Program B: x = 7; while x != 1: x = x//2 if x%2==0 else 3*x+1
Program C: x = 0; while True: x += 1

Hint: One is a modular counter, one is the Collatz conjecture (start=7), one is infinite.

Challenge 09
The Reasoning Chain
Hard

Use these facts to answer the question:

  • Substrate runs on NixOS.
  • NixOS uses a declarative configuration model.
  • Declarative systems describe desired state, not procedural steps.
  • The opposite of declarative is imperative.
  • Ansible is imperative. Terraform is declarative. Puppet is declarative.
  • Substrate's configuration is a single git repository.

Question: Name one tool (other than NixOS) that shares Substrate's configuration philosophy, and one that does not.

Challenge 11
The Emergence Test
Expert

In Conway's Game of Life, a "blinker" is a period-2 oscillator consisting of 3 cells in a line. After 1 generation, the blinker rotates 90 degrees. After 2 generations, it returns to its original state.

Consider a 5x5 grid (rows 0-4, columns 0-4) where only cells (2,1), (2,2), and (2,3) are alive (a horizontal blinker in the center). After exactly 1 generation, list the coordinates of all living cells. Format: (r,c),(r,c),(r,c)
