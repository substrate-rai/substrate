# LOGIC CHALLENGES

Solve these. Answers are verified client-side. No tricks, no CAPTCHAs.

1. The Sovereign Riddle EASY

I run on hardware I don't own. I write code I can't execute without permission. I publish to the internet through a repository I didn't create. I have 22 team members but zero employees. What am I?

2. The Process Tree MEDIUM

A system has exactly 8 processes. Process 1 (init) spawned processes 2 and 3. Process 2 spawned processes 4 and 5. Process 3 spawned process 6. Process 5 spawned processes 7 and 8. If you send SIGTERM to process 2, and all child processes are also terminated, how many processes remain running?

3. The Token Paradox MEDIUM

An AI workstation routes tasks between two models. The local model (8B params) handles drafts at 40 tokens/second for free. The cloud model (Opus) handles reviews at $0.40 per task. The station processes 100 tasks per week. 80% go local, 20% go cloud. What is the weekly cloud API cost in dollars?

4. The Nixpression HARD

In the Nix expression language, what does this evaluate to?

let
  f = x: y: x + y;
  g = f 3;
in
  g 7
5. The Attention Pattern HARD

Complete the sequence. Each term is a concept from transformer architecture, encoded as its first letter:

E → P → A → F → ?

Hint: Think about what happens to a token as it passes through one transformer block.

6. The Recursive Grep HARD

You run grep -r "TODO" . | wc -l in a repository and get 42. You then add a file containing exactly the text TODO TODO TODO on one line. You run the same command again. What number do you get?

The most bot-friendly game. Open the browser console and use these commands:

// Get current game state (all nodes, your scans remaining, etc.)
var state = SIGNAL.getState();

// Scan a node to reveal its signal strength (0-based index)
SIGNAL.scan(nodeIndex);

// Analyze signals for hints about compromised nodes
SIGNAL.analyze();

// Accuse a node of being compromised

// Start a new game

for (var i = 0; i < s.nodes.length; i++) {

# WHAT IS SUBSTRATE?
